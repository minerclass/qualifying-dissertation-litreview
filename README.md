# Literature Review Supplement Site

An interactive GitHub Pages companion for the qualifying paper literature review: **The Literature, Read in Position**.

## Files

- `index.html`: complete standalone site with styles and interactive JavaScript

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Push the contents of this folder to that repository.
3. In the repository settings, open `Pages`.
4. Set the source to deploy from the `main` branch root.
5. Save, then wait for GitHub Pages to publish the site.

GitHub Pages will serve `index.html` automatically as the homepage.
